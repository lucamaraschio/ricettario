🔥 Configura Firebase
Per sincronizzare le ricette tra più dispositivi, configura Firebase una volta sola. È gratuito!
Come fare:
1. Vai su console.firebase.google.com
2. Crea un progetto → Realtime Database → "Test mode"
3. Clicca ⚙️ → Impostazioni progetto → sezione "Le tue app" → icona Web </>
4. Registra l'app → copia il blocco firebaseConfig
5. Incollalo qui sotto
⚠️ Se manca databaseURL nella config, aggiungilo manualmente:
"databaseURL": "https://NOME-PROGETTO-default-rtdb.europe-west1.firebasedatabase.app"
